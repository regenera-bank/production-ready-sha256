# Regenera Bank — Risk & Control

**Documento:** RISK-CONTROL-BASELINE-001  
**Estado:** baseline técnica verificável  
**Responsável declarado:** Don Paulo Ricardo  
**Assinatura criptográfica:** pendente de chave externa real

Risco não é opinião.
É decisão com motivo, evidência e consequência.

Este pacote entrega regras determinísticas e testáveis para fraude, AML, KYC,
sanções, crédito, reconciliação, contabilidade, casos, auditoria e reporte.

Não entrega homologação com lista oficial, bureau, HSM, IAM, SIEM ou regulador.
Sem integração externa comprovada, o estado correto é bloqueado.
Não é produção por declaração.
É produção quando a evidência fecha.

## Comandos

```bash
make all
```

O comando valida a árvore, executa os testes, procura segredos, gera evidência
e verifica o manifesto.

## Limite institucional

A biblioteca é uma implementação de referência operacional.
Ativação exige integração, aprovação independente, commit assinado e exercício
com dados e sistemas autorizados.

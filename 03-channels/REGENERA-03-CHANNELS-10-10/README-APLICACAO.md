# Aplicação do pacote 03-channels

O diretório `03-channels` substitui a fronteira homônima no workspace canônico.

Antes de aplicar:

1. valide o SHA-256 externo;
2. execute `make all` na árvore extraída;
3. preserve backup do destino atual;
4. revise o diff;
5. assine o ZIP com chave real;
6. obtenha aprovação independente.

Não copie `dist` para o repositório.
`dist` é prova local de execução.
Código ativo fica em `03-channels`.

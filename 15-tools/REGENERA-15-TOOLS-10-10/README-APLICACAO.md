# Aplicação

1. Extraia o pacote em diretório vazio.
2. Execute `make all`.
3. Revise `evidence/` e `governance/RELEASE-APPROVAL.json`.
4. Use a CLI com `PYTHONPATH=src python3 -m regenera_tools.cli`.
5. Não habilite operação destrutiva sem ticket, revisor independente e backup validado.
6. Assine o ZIP externamente com a chave institucional real.

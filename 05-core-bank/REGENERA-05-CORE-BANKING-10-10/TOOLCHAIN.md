# Ferramental de validação

| Componente | Versão mínima | Função |
|---|---:|---|
| Java | 21 | execução da bateria e consumo do artefato compilado |
| Kotlin | 1.9 | compilação do domínio |
| Python | 3.11 | validação, segurança, build e integridade |
| Make | 4 | coordenação dos gates locais |

O pacote não instala dependências.
Ferramenta ausente interrompe a execução.
Pipeline verde por ausência de comando é fraude operacional.

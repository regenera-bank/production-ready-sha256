# Requisito de commit assinado

O commit de incorporação desta release precisa ser assinado pela chave real do responsável.

A evidência mínima inclui:

- fingerprint da chave;
- commit hash;
- resultado de `git verify-commit`;
- identidade do aprovador independente;
- referência da revisão;
- hash do ZIP distribuído.

Este pacote não contém commit fabricado nem assinatura simulada.

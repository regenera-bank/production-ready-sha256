# Aplicação do pacote

1. Extraia o ZIP em diretório vazio.
2. Execute `make all` com Python 3.11 ou superior.
3. Verifique `evidence/integrity/PAYLOAD-CHECKSUMS.sha256`.
4. Vincule os adaptadores externos somente depois de homologação formal.
5. Gere a assinatura `.asc` com a chave privada institucional real.

## Bloqueios de produção

A release não contém credenciais, certificados, protocolos, resultados de desastre, aceite de auditoria ou homologação de terceiros. Esses itens permanecem pendentes até serem produzidos pelo ambiente responsável.

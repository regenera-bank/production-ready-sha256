# Runbook — Documento Vencido

1. Marque o documento como não confiável para decisão nova.
2. Notifique owner e reviewers.
3. Revalide conteúdo e dependências.
4. Atualize revisão ou altere estado para `SUPERSEDED`/`RETIRED`.
5. Execute `make all`.

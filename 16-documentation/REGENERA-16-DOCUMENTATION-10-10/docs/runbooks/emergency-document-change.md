# Runbook — Mudança Documental Emergencial

1. Registre incidente ou mudança emergencial.
2. Limite o escopo.
3. Preserve o conteúdo anterior.
4. Exija revisão independente posterior com prazo definido.
5. Não use urgência para inventar aprovação, protocolo ou evidência.

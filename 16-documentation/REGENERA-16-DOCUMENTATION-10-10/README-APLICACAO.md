# Aplicação do pacote

1. Extraia o ZIP em diretório vazio.
2. Execute `make all` com Python 3.11 ou superior.
3. Confirme `PASS` em validação, testes, segurança, build e integridade.
4. Compare o hash externo do ZIP.
5. Gere a assinatura `.asc` somente com a chave privada institucional real.
6. Vincule a release documental ao commit assinado que a originou.

Este pacote não deve ser publicado como evidência de produção sem aprovação humana independente e assinatura criptográfica externa.
